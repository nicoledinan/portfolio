Climate Change Data

Javascript Project

Project Summary

For this JavaScript project, I aimed to bring to light the issue of climate change by creating a mobile web app that displays the changes in average global temperature. The charts are created using Chart.js and JavaScript and the layout of the site is coded with HTML and CSS. This site was intended to be viewed on a mobile device, but also works for larger screen sizes.

My Role

Concept Sketch

Using data from the National Oceanic and Atmospheric Administration, I began to develop a story and thinking about how to approach displaying the data in a way that would be understandable. I created a few sketches to work out a basic idea for what I needed to create.

Wireframe & Prototype

After I had a good idea of how I wanted to display the data, I created wireframes in Figma. From there, I further developed the design of the project and created prototypes.
